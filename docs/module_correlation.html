abstract calculate(estimator: str) → Tuple[ndarray, ndarray, ndarray]

Calculates the angular correlation function.

Peebles & Hauser estimator (1974):

\[\hat{\xi}=\frac{DD}{RR}-1\]

1st Landy & Szalay estimator (1993):

\[\hat{\xi}=\frac{DD-2DR-RR}{RR}\]

2nd Landy & Szalay estimator (1993):

\[\hat{\xi}=\frac{DD-DR-RD-RR}{RR}\]
Parameters:
estimatorstr

Estimator being considered (“dd”, “rr”, “dr”, “rd”, “peebles_hauser”, “landy_szalay_1”, “landy_szalay_2”).

Returns:
np.ndarray

The bin of angles \(\theta\) (in arcmins).

np.ndarray

The angular correlations function \(\xi(\theta)\).

np.ndarray

The angular correlation function errors \(\sigma_\xi(\theta)\).
