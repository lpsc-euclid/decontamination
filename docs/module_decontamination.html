class decontamination.decontamination.decontamination_elasticnet.Decontamination_ElasticNet(dim: int, dtype: <numeric type> = <class 'np.float32'>, rho: float = 1.0, l1_ratio: float = 0.5, alpha: float = 0.01, tolerance: float | None = None)

Bases: Decontamination_Abstract

Systematics decontamination using the Elastic Net method.

Parameters:
dimint

Dimensionality of the input data.

dtype<numeric type>, default: np.float32

Neural network data type, either np.float32 or np.float64.

rhofloat = 1.0

Constant that multiplies the penalty terms.

l1_ratiofloat = 0.5

Mixing parameter, with 0 <= l1_ratio <= 1. For l1_ratio = 0 the penalty is an L2 penalty. For l1_ratio = 1 it is an L1 penalty. For 0 < l1_ratio < 1, the penalty is a combination of L1 and L2.

alphafloat = 0.01

Learning rate.

tolerancefloat = None

The tolerance for the optimization.
