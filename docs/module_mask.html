Module decontamination.mask¶

This module provides primitives to perform projection from image to HEALPix masks.

HEALPix masks¶

decontamination.mask.healpix.build_healpix_wcs(wcs: astropy.wcs.WCS) → astropy.wcs.WCS¶

Adjusts the provided World Coordinate System (WCS) object to perform proper HEALPix projections.

Parameters:
wcsWCS

The original WCS object.

Returns:
WCS

The modified WCS object.

decontamination.mask.healpix.image_to_healpix(wcs: astropy.wcs.WCS, nside: int, footprint: ndarray, rms_image: ndarray, bit_image: ndarray | None = None, rms_selection: float = 10000.0, bit_selection: int = 0, show_progress_bar: bool = False) → Tuple[ndarray, ndarray, ndarray]¶

???

Parameters:
wcsWCS

The modified WCS object (see build_healpix_wcs()).

nsideint

The HEALPix nside parameter. Must be less or equal to 16384.

footprintnp.ndarray

HEALPix indices of the observed region. Nested ordering only.

rms_imagenp.ndarray

2-d image containing the RMS (aka. noise) information.

bit_imagenp.ndarray, default: None

2-d image containing the bit (aka. data quality) information.

rms_selectionfloat, default: 1.0e4

Reject the pixel if RMS == 0 or RMS > rms_selection.

bit_selectionint, default: 0x00

Reject the pixel if (bit & bit_selection) != 0x00.

show_progress_barbool, default = False

Specifies whether to display a progress bar.

Returns:
Tuple[np.ndarray, np.ndarray, np.ndarray]

First array contains the RMS (aka. noise) mask. Second array contains the bit (aka. data quality) mask. Third array contains the coverage (≡ fraction of observed sky) mask.
